This repository is the code and materials for the reproductivity of the manuscript "Advancing Freezing Rain Forecasting Up to 24 Hours: Harnessing Complex Network Priors in BiLSTM-Transformer Models" submitted to the journal JGR: Machine Learning and Computation.
Requirement:
1.torch >= 2.4.1+cu118 2.captum >=0.7.0 3.scikit-learn >=1.3.0

ResNet_BiLSTM_Transformer_seeded.ipynb is the code to split dataset, training, and validation for the baseline and source-enhanced models.

IG_interactions.ipynb is the code for the analysis of interpretability and attribution for the deep learning models.
